中山國小聯絡簿與學生簽到系統 v4 UI 強化版

使用方式：
1. 解壓縮 zip 檔。
2. 用 Chrome 或 Edge 開啟 index.html。
3. 建議點右上角「全螢幕」，搭配教室觸控式大螢幕使用。

本版修改：
- 上方時間顯示區域重新美化，比例更接近參考圖。
- 學生簽到區只顯示座號，不顯示姓名。
- 聯絡簿每日預設內容改為空白。
- 新增聯絡簿字體放大、縮小、恢復預設按鈕。
- 每日聯絡簿與簽到紀錄仍會自動儲存在本機瀏覽器。

注意：
資料儲存在同一台電腦、同一個瀏覽器的 localStorage。
若換電腦或清除瀏覽器資料，紀錄可能消失，建議定期匯出 CSV 備份。
